# 노션 기반 견적서 공유 시스템 MVP PRD

## 🎯 핵심 정보

**목적**: Notion에서 입력한 견적서를 클라이언트가 웹 링크로 조회하고 PDF로 다운로드받을 수 있는 플랫폼 구축
**사용자**: 견적서를 작성하는 업체(발행자)와 견적서를 확인하는 고객(수신자)

---

## 🚶 사용자 여정

```
[시작] 클라이언트가 견적서 링크 수신
   ↓

[공개 견적서 조회 페이지]
   ├─ 견적서 데이터 로드 (Notion API 연동)
   ├─ 견적서 정보 표시
   └─ PDF 다운로드 버튼 제시

   ↓

[조건 분기]
   ├─ [PDF 다운로드 클릭]
   │  └─ 브라우저 기본 다운로드 실행
   │     (브라우저 다운로드 폴더에 PDF 저장)
   │
   └─ [링크 공유 클릭]
      └─ 견적서 링크 복사
         (클립보드에 링크 저장)
```

## 📝 구현 노트

### Notion API 통합 방식

1. **환경변수 설정**: Notion API Key와 Database ID를 `.env.local`에 저장
2. **서버 액션**: Next.js Server Actions를 통해 서버 사이드에서 Notion 데이터 조회
3. **캐싱 전략**: 견적서 데이터를 ISR (Incremental Static Regeneration)로 캐시하여 성능 최적화

### PDF 다운로드 구현

- **라이브러리**: `html2pdf.js` 또는 `@react-pdf/renderer` 사용
- **방식**: 클라이언트 사이드에서 현재 페이지를 PDF로 변환하여 다운로드
- **파일명**: `{invoiceNumber}.pdf` (예: `INV-2025-001.pdf`)

### URL 구조

- **홈페이지**: `/`
- **공개 견적서 조회**: `/invoice/[invoiceId]` (동적 라우팅)
